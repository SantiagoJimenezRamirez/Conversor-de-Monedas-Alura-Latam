Bienvenidos al Conversor de Monedas Alura Latam.

Este Proyecto fue creado por : Santiago Jimenez Ramirez y lo puedes encontrar en mi Github.

Explicacion:

Con el Lenguaje de Programacion Python se ha hecho Web Scrapping a las paginas oficiales de Google Finance para tomar los datos de las monedas las cuales se requeria hacer la conversion; Luego hemos puesto estos datos en un formato Json para que en Java podamos tomar estos datos y usarlos a nuestro gusto.

Cabe aclarar que para poder leer estos datos en Java fue necesario hacer una Conexion para que este formato Json se subiera a la red y mediante el Link (http://127.0.0.1:5000/data) se empezaran a extraer dichos datos.

Requerimientos:

Fue necesario Instalar Librerias como:
	En Python:
		(BeautifulSoup, requests, json, re)
	En Java:
		(Json (link de Descarga:"https://mvnrepository.com/artifact/org.json/json/20220320"))

Recomnedaciones: 
	Abra por aparte las Carpetas si es posible en Eclipse ejecute los documetos Java y desde la misma consola ejecute los archivos Python.
